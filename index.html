Tus finanzas bajo control, en tu propio proyecto Supabase

¿Por qué usar Registro Contable?

Diseñado para ser rápido, privado y completamente configurable.

🛡️

Privacidad Absoluta

Tus datos financieros nunca pasan por servidores de terceros. Todo se almacena de forma segura en tu propio proyecto de Supabase.

⚡

Carga Ultrarrápida

Construido como una Single Page Application optimizada, sin librerías pesadas, para que abras y registres tus gastos al instante.

📊

9 Gráficos Interactivos

Analiza visualmente la distribución de tus gastos, compara ingresos vs gastos, controla tus presupuestos y ve tendencias interanuales.

📅

Control de Presupuestos

Establece límites mensuales por categoría para no gastar de más, y calcula de forma automática cuánto dinero te sobra cada mes.

🔄

Acciones Automatizadas

Transfiere de forma automática los presupuestos sobrantes de tus categorías principales directamente hacia tu categoría de Ahorro.

📱

Diseño Responsive

Accede cómodamente desde tu móvil, tablet u ordenador. La interfaz se adapta perfectamente a cualquier tamaño de pantalla.

Configuración en 3 Sencillos Pasos

1

Crea tu Base de Datos

Crea un proyecto gratuito en Supabase donde se guardarán todos los registros, transacciones y presupuestos.

2

Obtén tus Credenciales

Copia la URL de la API y la clave pública anónima (Anon Key) desde el panel de control de tu proyecto de Supabase.

3

Conéctate y Sincroniza

Pega tus credenciales en nuestra aplicación web para iniciar la sincronización de tus datos en tiempo real.
